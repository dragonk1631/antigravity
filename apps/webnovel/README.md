# 🌌 Nexus Wiki - 웹소설 세계관 관리 도구

**Nexus Wiki**는 웹소설 작가님들이 복잡한 인물 관계와 세계관 설정을 직관적으로 관리할 수 있도록 돕는 웹 애플리케이션입니다.
**컴퓨터 지식이 없어도**, 브라우저 상에서 바로 세계관을 만들고 저장할 수 있습니다.

## ✨ 주요 기능

### 1. 🕸️ 인터랙티브 인물 관계도

- **시각화**: 등장인물, 장소, 사건들이 어떻게 연결되어 있는지 한눈에 파악할 수 있습니다.
- **물리 엔진**: D3.js 기반의 물리 엔진이 적용되어, 노드들이 겹치지 않고 자연스럽게 배치됩니다.
- **자유로운 조작**: 드래그로 위치 이동, 마우스 휠로 줌/팬이 가능합니다.

### 2. ✏️ 간편한 편집 (No-Code)

- **편집 모드**: 하단의 '편집 모드' 버튼을 눌러 수정을 시작하세요.
- **노드 추가**: '노드 추가' 버튼으로 새로운 인물이나 사건을 생성할 수 있습니다.
- **상세 정보 수정**: 노드를 클릭하면 나타나는 사이드바에서 이름, 설명, 이미지 주소 등을 바로 수정할 수 있습니다.
- **관계 연결**: 사이드바 하단에서 다른 인물을 선택하여 관계(친구, 적대 등)를 맺거나 끊을 수 있습니다.

### 3. 💾 데이터 저장 및 백업

- **자동 저장**: 작업한 내용은 **브라우저에 자동으로 저장**됩니다. 창을 닫아도 사라지지 않습니다.
- **파일 백업**: '저장' 버튼을 눌러 데이터를 파일(`.json`)로 내려받을 수 있습니다. 다른 컴퓨터로 옮기거나 안전하게 보관하세요.
- **불러오기**: '열기' 버튼으로 저장해둔 파일을 다시 불러올 수 있습니다.

## 🚀 시작하기 (Getting Started)

### 필수 조건

- [Node.js](https://nodejs.org/) (설치만 되어 있으면 됩니다)

### 설치 및 실행

1. 터미널(명령 프롬프트)을 열고 프로젝트 폴더로 이동합니다.

   ```bash
   cd apps/webnovel
   ```

2. 처음에만 한 번 실행합니다 (설치):

   ```bash
   npm install
   ```

3. 앱 실행하기:

   ```bash
   npm run dev
   ```

4. 인터넷 브라우저 주소창에 `http://localhost:5173`을 입력하면 실행됩니다.

## 🛠️ 기술 스택 (개발자용)

- **Core**: React, TypeScript, Vite
- **Visual**: D3.js (Force-Directed Graph)
- **State**: Zustand + Persist Middleware
- **Style**: Tailwind CSS v4
