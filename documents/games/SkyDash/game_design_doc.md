# SkyDash - 게임 품질 향상 기획서 (Game Design Document)

## 1. 개요

이 문서는 SkyDash의 전반적인 품질과 완성도를 높이기 위한 기획 내용을 기록하고 관리하는 문서입니다. 회의를 통해 결정된 주요 기능, 비주얼 피드백, 오디오 개선 사항 등을 아카이브합니다.

---

## 2. 품질 향상을 위한 브레인스토밍 (Seed Ideas)

### 2.1 비주얼 이펙트 (VFX)

- **착지 파티클**: 캐릭터가 계단에 착지할 때 작은 먼지나 불꽃 파티클이 발생하여 타격감 강화.
- **화면 흔들림 (Screen Shake)**: 높은 콤보를 달성하거나 빠른 속도로 오를 때 미세한 화면 흔들림 효과 추가.
- **속도감 연출**: 일정 콤보 이상 도달 시 뒷배경에 속도선(Speed Lines) 효과 추가.

### 2.2 오디오 및 사운드 (SFX)

- **콤보 사운드**: 콤보가 올라갈 때마다 피치가 높아지는 사운드 피드백 제공.
- **환경 음향**: 고도가 높아짐에 따라 바람 소리나 대기 환경 사운드가 변화하는 연출.
- **성취감 사운드**: 100계단 돌파나 신기록 달성 시 축하 사운드 추가.

### 2.3 게임플레이 깊이

- **피버 모드 (Fever Mode)**: 에너지가 가득 차거나 특정 조건을 만족하면 배경 색상이 변하며 점수 배율이 증가하는 화려한 연출.
- **방해 요소**: 가끔 흔들리는 계단이나 가시가 튀어나오는 등 긴장감을 주는 요소 검토.

### 2.4 사용자 경험 (UX) 및 연출

- **씬 전환 효과**: 메인 메뉴에서 게임 시작 시 더욱 부드럽고 세련된 페이드/줌 트랜지션 적용.
- **UI 애니메이션**: 버튼 클릭이나 점수 갱신 시 더욱 '찰진(Juicy)' 애니메이션 효과 부여.
