# SkyDash - 게임 품질 향상 기획서 (Game Design Document)

## 1. 개요

이 문서는 SkyDash의 전반적인 품질과 완성도를 높이기 위한 기획 내용을 기록하고 관리하는 문서입니다. 회의를 통해 결정된 주요 기능, 비주얼 피드백, 오디오 개선 사항 등을 아카이브합니다.

---

## 2. 품질 향상을 위한 브레인스토밍 (Seed Ideas)

## 2. 품질 향상을 위한 상세 명세 및 우선순위 (Detailed Specs & Priority)

### 2.1 비주얼 이펙트 (VFX)

| 항목 | 상세 기술 명세 | 우선순위 |
| :--- | :--- | :--- |
| **착지 파티클** | Phaser.GameObjects.Particles 활용. 계단 좌표에 맞춰 `gravityY`, `lifespan` 조절된 먼지 효과 생성. | **High** |
| **화면 흔들림** | `scene.cameras.main.shake(duration, intensity)` 활용. 콤보 배수 및 속도에 따라 강도 차등 적용. | **High** |
| **속도감 연출** | `Phaser.GameObjects.TileSprite`를 이용한 배경 레이어링 + 콤보 시 `Alpha` 페이드인 되는 Speed Lines 구현. | **Medium** |

### 2.2 오디오 및 사운드 (SFX)

| 항목 | 상세 기술 명세 | 우선순위 |
| :--- | :--- | :--- |
| **콤보 사운드** | `SoundManager.playClimb(pitch)` 호출 시 콤보 누적치에 비례하여 `pitch` 값(1.0 ~ 2.0) 증가 로직 적용. | **High** |
| **환경 음향** | 고도(Score)에 따라 Web Audio `BiquadFilterNode`를 활용하여 Low-pass 필터 가청화 (바람 소리 효과). | **Medium** |
| **성취감 사운드** | 특정 마일스톤 도달 시 아르페지오 형태의 축하 멜로디 시퀀싱 재생. | **Medium** |

### 2.3 게임플레이 및 시스템

| 항목 | 상세 기술 명세 | 우선순위 |
| :--- | :--- | :--- |
| **피버 모드** | 에너지 완충 시 진입. 배경 틴트 변경, 점수 2배, 전용 BGM 레이어(필터 오픈) 추가. | **Medium** |
| **레벨 시스템** | 50계단마다 배경 테마(색상, 타일셋) 변경 및 에너지 감소 속도 단계적 상승. | **High** |

### 2.4 비주얼 아이덴티티: 리얼 픽셀 아트

| 항목 | 상세 기술 명세 | 우선순위 |
| :--- | :--- | :--- |
| **픽셀 렌더링** | `pixelArt: true` 설정 확인 및 모든 스프라이트의 `texture.setFilter(Phaser.Textures.FilterMode.NEAREST)` 보장. | **High** |
| **비트맵 폰트** | `.ttf`를 직접 사용하기보다 픽셀 스타일에 최적화된 `.xml` 기반 XML Bitmap Font 적용으로 가독성 극대화. | **High** |

## 3. 회의 및 결정 사항 기록

### 2026-01-27: Phase 1 계획 확정

- **VFX 우선순위**: 타격감(착지 파티클)과 화면 흔들림을 최우선으로 구현.
- **오디오 엔진 고도화**: 단순 `setTimeout` 방식에서 `Phaser.Time.TimerEvent` 또는 Web Audio API `currentTime` 기반의 정밀 스케줄링으로 전환 유도.
- **구조화**: 각 기능별(VFX, Audio, State) 매니저 클래스 역할 분담 명확화.
